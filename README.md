# 钱前问问

这是一个面向参赛交付的智能体 MVP 项目，围绕“大学生理财陪伴 AI 搭子”搭建。

当前核心思路是：用陪伴式、过来人式的对话语气，帮助大学生从零开始建立理财判断力，完成理财起点建档、热点新闻理解、小额理财计划和复盘，同时用风险护栏拦截贷款理财、带单、杠杆和高收益骗局。

## 核心定位

面向大学生的学长学姐型入门理财 AI 搭子，陪用户从“有点闲钱想试试”，逐步走向“知道钱能不能动、看得懂理财逻辑、能理解财经热点、敢于小额实践、也懂得复盘和避险”。

## MVP 核心能力

1. 我的理财起点  
   通过聊天式轻量问卷建立用户档案，识别资金来源、短期支出、应急备用金、理财经验、风险承受能力和理财动机。

2. 热点雷达  
   推送或解读金融、财经、重大、科技新闻，拆解其可能影响的变量、行业板块、利好利空倾向和新手观察点。

3. 小额计划与复盘  
   根据用户档案生成低门槛入门理财计划，并持续复盘收益、波动、手续费、情绪变化和冲动行为。

风险护栏贯穿全流程，不单独作为交易功能展示。

## 智能体技术路线

这个项目的正确方向不是“写死一堆固定回答”，而是：

```text
LLM 大模型：负责自然、灵活、有学长学姐感的对话
RAG 知识库：负责专业理财知识检索和可追溯回答
实时新闻工具：负责金融、财经、重大、科技热点获取
风险护栏规则：负责贷款、杠杆、带单、高收益骗局拦截
```

本地演示版已经提供简化后端 `server.py`：

- 有 `LLM_API_KEY` 时：走“大模型 + RAG”。
- 没有 `LLM_API_KEY` 时：走“知识库检索 + 规则兜底”。
- 新闻接口联网可用时：获取实时 RSS 新闻。
- 新闻接口不可用时：使用备用新闻保证演示不断。

## 如何体验原型

直接打开：

```text
app/index.html
```

这个页面是一个可演示的静态 MVP，用规则模拟智能体核心流程。正式参赛时，可以把 `agent/system_prompt.md` 和 `knowledge/` 里的内容复制到扣子、Dify、通义智能体、智谱智能体等平台，再发布二维码。

## 建议搭建平台

优先选择能支持以下能力的平台：

- 自定义智能体人设和系统提示词
- 上传知识库文档
- 配置工作流或快捷指令
- 支持网页体验链接或二维码
- 最好支持联网搜索或定时新闻推送

如果平台不支持联网新闻，MVP 可以使用“用户粘贴新闻，AI 进行热点雷达分析”的方式完成演示。
