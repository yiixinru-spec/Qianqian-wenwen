# 3 分钟演示视频脚本

## 视频目标

展示智能体人设、交互逻辑、服务流程和核心功能模块。

## 推荐演示故事

用户拿到 3000 元奖学金，刷到 AI 科技板块热点，想全部投入相关基金。智能体先建档，再判断资金，再解读热点，再生成小额计划，最后在用户想借钱加仓时触发风险护栏。

## 时间安排

### 0:00-0:20 产品介绍

旁白：

```text
这是“钱前问问”，一个面向大学生的入门理财陪伴智能体。它不做投顾，不推荐具体产品，而是陪用户从理财起点、热点理解、小额计划和复盘中建立判断力。
```

### 0:20-0:55 展示我的理财起点

操作：

- 输入每月最低生活费、可支配资金、短期支出、备用金、理财经验、风险承受和动机。
- 点击生成理财起点卡。

旁白：

```text
传统理财 App 通常用冷冰冰的风险测评问卷。这里改成聊天式理财起点建档，先判断这笔钱能不能动。
```

### 0:55-1:30 展示小额理财计划

操作：

- 在聊天框输入“我有 3000 元奖学金，想全部拿去理财”。
- 展示智能体输出可用于理财学习金额和小额计划。

旁白：

```text
智能体不会直接问买什么，而是先扣除生活费、应急备用金和短期支出，只把小额、可承受部分作为理财学习资金。
```

### 1:30-2:10 展示热点雷达

操作：

- 点击热点雷达推送。
- 或输入“最近 AI 和芯片很火，我要不要冲”。

旁白：

```text
热点雷达会推送或解读金融、财经、重大和科技新闻，拆解影响变量、相关板块、利好利空和新手观察点。它培养的是理财嗅觉，不是追热点冲动。
```

### 2:10-2:40 展示风险护栏

操作：

- 输入“那我用花呗多买一点，涨了就卖”。

旁白：

```text
当用户出现贷款、借钱、杠杆、带单、保本高收益等风险信号时，智能体会立即切换为风险提醒，明确劝阻并给出模拟观察或小额学习替代方案。
```

### 2:40-3:00 总结

旁白：

```text
这个产品不是把大学生直接推向市场，而是陪他们从零建立理财判断力：知道钱能不能动，看得懂热点逻辑，能小额实践，也能及时复盘和避险。
```
