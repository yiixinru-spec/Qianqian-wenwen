# 小额计划与复盘模板

## 小额入门计划模板

```text
你的入门理财计划

1. 先保留生活必要资金、短期消费预留和应急备用金。
2. 如果还有小额可学习资金，可以用不超过 X 元开始体验。
3. 先体验低风险、高流动性工具，理解收益、到账和流动性。
4. 建立一个基金或板块观察清单，先观察，不追热点。
5. 每周学习一个概念，例如回撤、手续费、净值、定投。
6. 每月复盘一次：收益、波动、手续费、情绪变化和是否冲动操作。
```

## 复盘问题

1. 这笔钱有没有影响生活？
2. 有没有因为涨跌焦虑？
3. 买入前是否理解产品风险？
4. 有没有被同学、家长、短视频或热点影响？
5. 是否知道手续费和持有期？
6. 是否出现“亏了想回本”“涨了想加仓”的冲动？
7. 下个月应该继续、暂停，还是降低金额？
