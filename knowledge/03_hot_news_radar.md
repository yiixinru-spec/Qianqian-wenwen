# 热点雷达新闻解读框架

## 功能定位

热点雷达不是荐股工具，而是帮助大学生看懂金融、财经、重大、科技新闻背后的理财逻辑，培养“理财嗅觉”。

## 可覆盖新闻类型

1. 金融新闻：利率、汇率、债券、黄金、股市波动。
2. 财经政策：消费刺激、房地产政策、产业扶持、监管政策。
3. 重大新闻：国际冲突、能源价格、自然灾害、公共事件。
4. 科技新闻：AI、芯片、算力、机器人、新能源车、半导体、医药创新。

## 标准输出结构

```text
热点雷达拆解
一句话翻译：
影响变量：
相关板块：
利好/利空倾向：
影响周期：
新手观察点：
风险提醒：
```

## 常见影响链条

### AI / 芯片 / 算力

影响变量：技术预期、算力需求、产业链订单、市场情绪。

相关板块：半导体、算力、云服务、AI 应用、机器人。

提醒：科技热点波动大，利好行业不等于所有相关基金都会涨。

### 降息 / 降准 / 利率变化

影响变量：资金成本、市场流动性、债券价格、估值。

相关板块：债券、银行、地产、成长型行业、黄金。

提醒：利率变化影响复杂，不同资产方向可能不同。

### 消费政策

影响变量：消费意愿、收入预期、政策刺激。

相关板块：食品饮料、旅游、零售、电商、餐饮。

提醒：短期政策刺激不等于长期基本面改善。

### 油价 / 国际冲突 / 避险情绪

影响变量：能源成本、供应链、通胀预期、避险需求。

相关板块：石油、黄金、航空、物流、化工、农业。

提醒：重大事件下市场波动可能很剧烈，新手不适合因为情绪消息追高。

### 新能源

影响变量：政策支持、需求预期、原材料价格、产能竞争。

相关板块：新能源车、锂电池、光伏、储能。

提醒：销量增长不等于企业利润一定增长，还要看价格竞争和库存。
